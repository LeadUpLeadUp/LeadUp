# מפרט הנחות – חברות הביטוח (אתר חיצוני)

זה אתר סטטי (HTML/CSS/JS) שנועד לעלות לריפו נפרד ולהיפתח מתוך ה‑CRM בלחיצה על כפתור "מפרט פוליסות עדכני של חברות הביטוח".

## איך מעלים ל‑GitHub Pages (הכי קל)
1. צור ריפו חדש ב‑GitHub (לדוגמה: `policy-spec`).
2. העלה **את כל התיקייה הזו** (כולל `assets/docs`).
3. ב‑GitHub: Settings → Pages  
   - Source: Deploy from a branch  
   - Branch: main / root
4. קבל כתובת כמו: `https://<username>.github.io/policy-spec/`

## חיבור ל‑CRM
ב‑`app.js` של המערכת שלך תשים את הכתובת הזו כ‑URL.

## מה יש בפנים
- כפתור לכל חברה
- חלוקה לפי קטגוריות: ריסק / בריאות / מחלות קשות / משכנתא
- צפייה ב‑PDF/תמונות
- אזור אופציונלי שמקבל "הנחות" מהמערכת ב‑postMessage (לא חובה)
